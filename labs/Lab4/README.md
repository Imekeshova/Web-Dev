# Lab 4 - Angular Online Store

## Task 1
Completed official Angular tutorial.

## Task 2
Online store application that displays products from kaspi.kz.

## Features
- Product list using Angular components
- Product card component
- Data binding (interpolation, property binding, event binding)
- Share functionality (WhatsApp / Telegram)
- External links to kaspi.kz

## How to run

1. Install dependencies:
npm install

2. Run the project:
ng serve

3. Open in browser:
http://localhost:4200
