# Lab 5 – Online Store (Angular)

## Description
This project is an Online Store built with Angular.

Features:
- Categories (Smartphones, Accessories, Headphones, Other)
- Product list filtering by category
- Like button for each product
- Delete product functionality
- Share via WhatsApp and Telegram
- Component structure:
  - AppComponent
  - ProductListComponent
  - ProductItemComponent

## How to run

1. Install dependencies:
   npm install

2. Run the project:
   ng serve

3. Open in browser:
   http://localhost:4200

## Notes
- node_modules is not included
- Built using Angular standalone components
